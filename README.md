

Wizards of Destiny
--

Developed in 2024.

C++ code only; other assets were removed from this repository.

The game was developed with Unreal Engine C++. 

Wizards duel and battle creepy creatures. Wizards can be equipped with magic wands, potions, and flying brooms.

Single-player mode is implemented as an open-world mode where the player can collect supplies, unlock/discover secrets, and complete missions.

Multiplayer mode is implemented as an area duel mode, where players connect with each other to duel against themselves.
