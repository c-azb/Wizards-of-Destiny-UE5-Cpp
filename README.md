
Wizards of Destiny
--

Developed in 2024.

C++ code only, other assets were removed from this repository.

Game developed with Unreal Engine C++. 

Wizard duels and creepy creatures battles. Wizards can be equiped by magic wands, potions and flying brooms.

Single player mode implemented as a open world mode where the player can collect supplies, unlock/discover secrets, complete missions.

Multiplayer mode implemented as an area duel mode, where players connect with each other to duel against themselves.
